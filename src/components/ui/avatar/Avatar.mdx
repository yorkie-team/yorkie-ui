## Overview

An `Avatar` is a visual representation of a user, usually in the form of a profile picture, initials, or icon that represents the user.
It serves as a visual identifier for users, making it easier to distinguish between different individuals in the digital environment.

## Usage

An `Avatar` object consists of some optional properties:
- `size`
- `src`
- `name`

The following code shows how you can use the `Avatar` component:

<Source
  format={true}
  code={`
    import { Avatar } from '@/components/ui';

    <Avatar size="sm" src="https://i.pravatar.cc/300" name="John Doe" />
  `}
/>

## Examples

### Sizes

The default value for the `size` prop is `md`. You can use the `size` prop to modify the avatar's dimensions, with six available options: `xs`, `sm`, `md`, `lg`, xl, and `2xl`.

### Src

To import the avatar image, you can use the `src` prop.

If not provided, the avatar will be displayed by the `name` prop if provided or the user icon by default.

<Canvas
  of={AvatarStories.AvatarMd}
  source={{
    format: true,
    code: `<Avatar src="https://i.pravatar.cc/300" name="John Doe"} />`,
  }}
/>

### Name

By utilizing the `name` prop, you can import the name for the avatar. It will be displayed if the `src` prop is omitted.

<Canvas
  of={AvatarStories.WithNameOnly}
  source={{
    format: true,
    code: `<Avatar name="John Doe"} />`,
  }}
/>

### With No User Data

If the `src` and `name` props are omitted, the avatar will display with `UserIcon` by default.
