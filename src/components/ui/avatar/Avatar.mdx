## Overview

A graphical representation of the user, often used in profile sections.

## Usage

There are 6 sizes: `xs`, `sm`, `md` (default), `lg`, `xl`, `2xl`.

If `AvatarImage`'s src is error or undefined then `AvatarFallback` will be displayed by default.

<Source format={true} code={`
  <Avatar size="md">
    <AvatarFallback>PA</AvatarFallback>
    <AvatarImage src="https://i.pravatar.cc/300" alt="avatar" />
  </Avatar>
`} />
