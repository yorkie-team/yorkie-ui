## Usage

A `Avatar` consists of 3 properties:
- `size`
- `src`
- `name`

The following code shows how you can use the `Avatar` component:

<Source
  format={true}
  code={`
    import { Avatar } from '@/components/ui';

    <Avatar size="sm" src="https://i.pravatar.cc/300" name="John Doe" />
  `}
/>
