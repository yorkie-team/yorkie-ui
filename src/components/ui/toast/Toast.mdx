## Overview

Get real-time feedback on your actions with our on-screen messages.
To use the Toast component, set it up with createToaster hook. This hook manages grouping and placement of toasts. Use toast.create method to display a notification.

<Canvas
  of={ToastStories.Overview}
  source={{
    format: true,
    code: `
      const [Toaster, toast] = createToaster({
        placement: 'top-end',
        render(toast) {
          return (
            <Toast.Root>
              <Toast.Title>{toast.title}</Toast.Title>
              <Toast.Description>{toast.description}</Toast.Description>
              <Toast.CloseTrigger asChild>
                <CloseIcon />
              </Toast.CloseTrigger>
            </Toast.Root>
          );
        },
      });

      return (
        <>
          <Button variant="outline" onClick={() => toast.create({ title: 'Title', description: 'Description' })}>
            Create Toast
          </Button>
          <Toaster />
        </>
      );
    `,

}}
/>

## Examples

### [Configuring Toast](#configuring-toast)

To effectively configure the Toast component, it is imperative that you pass diverse options to the toast.create method. These options include, but are not limited to, `title`, `description`, `type`, `duration`, and `removeDelay`.

<Canvas
  of={ToastStories.ConfiguringToast}
  source={{
    format: true,
    code: `
      const [Toaster, toast] = createToaster({
        placement: 'bottom-start',
        render(toast) {
          return (
            <Toast.Root>
              <Toast.Title>{toast.title}</Toast.Title>
              <Toast.Description>{toast.description}</Toast.Description>
              <Toast.CloseTrigger>Close</Toast.CloseTrigger>
            </Toast.Root>
          )
        },
      })

      return (
        <>
          <Button
            onClick={() =>
              toast.create({
                title: 'Success',
                description: 'This is a success toast',
                type: 'success',
                duration: 20000,
                removeDelay: 250,
              })
            }
          >
            Toast
          </Button>
          <Toaster />
        </>
      )
    `,

}}
/>

### [Custom Rendered Toast](#custom-rendered-toast)

The Toast component provides the ability to use custom render functions, granting users greater flexibility in rendering content.

<Canvas
  of={ToastStories.CustomRenderedToast}
  source={{
    format: true,
    code: `
      const [Toaster, toast] = createToaster({
        placement: 'bottom-end',
        // custom render may go directly into the function below
        render(toast) {
          return (
            <Toast.Root>
              <Toast.Title>{toast.title}</Toast.Title>
              <Toast.Description>{toast.description}</Toast.Description>
              <Toast.CloseTrigger>Close</Toast.CloseTrigger>
            </Toast.Root>
          )
        },
      })

      return (
        <>
          <Button
            onClick={() =>
              toast.create({
                title: 'Please checkout',
                render: (toast) => (
                  <div>
                    {toast.title} <a href="https://yorkie.io">Yorkie UI</a>
                  </div>
                ),
              })
            }
          >
            Toast
          </Button>
          <Toaster />
        </>
      )
    `,

}}
/>
