## Spacing Tokens

Using a spacing system creates a better user experience and enables responsive design and customizable UI density, improving product accessibility and quality.

Apply use to value props: `mt`, `p`, `mb`, etc....

You can use negative numbers.

```jsx
<Box mt="-4" pd="10">
  Using Tokens
</Box>
```
