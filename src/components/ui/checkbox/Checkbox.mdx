## Overview

A control element that allows for multiple selections within a set.

<Canvas
  of={CheckboxStories.Overview}
  source={{
    format: true,
    code: `
      <Checkbox.Root>
        {(state) => (
          <>
            <Checkbox.Control>
              {state.isChecked && <CheckIcon />}
              {state.isIndeterminate && <MinusIcon />}
            </Checkbox.Control>
            <Checkbox.Label>Label</Checkbox.Label>
          </>
        )}
      </Checkbox.Root>
    `,
  }}
/>

## Usage

A `Checkbox` consists of some properties:
- `size`
- `defaultChecked`
- `checked="indeterminate"`
- `colorPalette`
- `disabled`

The following code shows how you can use the `Checkbox` component:

<Source format={true} code={`
  <Checkbox.Root>
    {(state) => (
      <>
        <Checkbox.Control>
          {state.isChecked && <CheckIcon />}
          {state.isIndeterminate && <MinusIcon />}
        </Checkbox.Control>
        <Checkbox.Label>Label</Checkbox.Label>
      </>
    )}
  </Checkbox.Root>
`} />
