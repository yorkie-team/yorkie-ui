## Overview

A collapsible component that displays content in a vertical stack.

<Canvas
  of={AccordionStories.Overview}
  source={{
    format: true,
    code: `
      const items = [
        { name: 'React', content: 'content of React item' },
        { name: 'Solid', content: 'content of Solid item' },
        { name: 'Svelte', content: 'content of Svelte item' },
        { name: 'Vue', content: 'content of Vue item' },
      ];

      <Accordion.Root>
        {items.map((item, id) => (
          <Accordion.Item key={id} value={item.name}>
            <Accordion.ItemTrigger>

## Usage

A `Accordion` consists of some properties:
- `multiple`
- `defaultValue`
- `disabled`

The following code shows how you can use the `Accordion` component:

<Source format={true} code={`
  import { Accordion } from '@/components/ui';

  const items = [
    { name: 'React', content: 'content of React item' },
    { name: 'Solid', content: 'content of Solid item' },
    { name: 'Svelte', content: 'content of Svelte item' },
    { name: 'Vue', content: 'content of Vue item' },
  ];

  <Accordion.Root {...args}>
      {items.map((item, id) => (
        <Accordion.Item key={id} value={item.name}>
          <Accordion.ItemTrigger>
            {item.name}
            <Accordion.ItemIndicator>
              <svg
                xmlns="http://www.w3.org/2000/svg"
                width="24"
                height="24"
                viewBox="0 0 24 24"
                fill="none"
                stroke="currentColor"
                stroke-width="2"
                stroke-linecap="round"
                stroke-linejoin="round"
              >
                <path d="m6 9 6 6 6-6"></path>
              </svg>
            </Accordion.ItemIndicator>
          </Accordion.ItemTrigger>
          <Accordion.ItemContent>
            <div>{item.content}</div>
          </Accordion.ItemContent>
        </Accordion.Item>
      ))}
    </Accordion.Root>
`} />
