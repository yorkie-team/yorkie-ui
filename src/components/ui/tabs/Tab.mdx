### [Lazy Mounting](#tab-lazy-mounting)

Lazy mounting is a useful feature that helps optimize the performance of a web page. It enables the content of a tab to be loaded only when the tab is first activated. This is particularly useful when the tab content is large or complex. To enable lazy mounting, you can make use of the lazyMount prop on the `Tabs.Content` component.

In addition, the `unmountOnExit` prop can be used in conjunction with lazyMount to unmount the tab content when the tab is deactivated, freeing up resources. The next time the tab is activated, its content will be re-rendered.

<Canvas
  of={TabStories.LazyMounting}
  source={{
    format: true,
    code: `
        <Tabs.Root lazyMount unmountOnExit>
          <Tabs.List>
            <Tabs.Trigger value="tab1">Content of Tab Pane 1</Tabs.Trigger>
            <Tabs.Trigger value="tab2">Content of Tab Pane 2</Tabs.Trigger>
            <Tabs.Trigger value="tab3">Content of Tab Pane 3</Tabs.Trigger>
            <Tabs.Indicator />

### [Disabled Tab](#tab-disabled-tab)

To disable a tab, you can pass the `disabled` prop to the `Tabs.Trigger` component. This will prevent the user from clicking or interacting with the tab.

  of={TabStories.DisabledTab}

### [Manual Activation](#manual-activation)

By default, the tab can be selected when it receives focus from either the keyboard or pointer interaction. This is called automatic tab activation.

In contrast, manual tab activation means the tab is selected with the

`Enter` key or by clicking on the tab
