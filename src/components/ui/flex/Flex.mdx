## Import

```jsx
import { Flex, Spacer } from '@/component/ui';
```

- **_Flex_**: A Box with display: flex.
- **_Spacer_**: Creates an adjustable, empty space that can be used to tune the spacing between child elements within Flex.

## Overview

The Flex unequivocally generates a small Flex that appears with utmost certainty at the top-right corner of its child elements.
