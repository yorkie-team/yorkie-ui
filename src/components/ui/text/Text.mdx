## Import


## Overview

Use the `as` property to specify an HTML tag. This changes the semantic meaning without affecting the visual style.

<Canvas
  of={TextStories.Overview}
  source={{
    format: true,
    code: `
      <div>
        <Text as="p">
          This is a <strong>p</strong> element.
        </Text>
        <Text as="label">
          This is a <strong>label</strong> element.
        </Text>
        <Text as="div">
          This is a <strong>div</strong> element.
        </Text>
        <Text as="span">
          This is a <strong>span</strong> element.
        </Text>
      </div>
    `,
  }}
/>

## Examples

### [Font Sizes](#text-font-sizes)

Use `size` or `textStyle` for text size include: `xs`, `sm`, `md`, `lg`, `xl`, `2xl`, `3xl`, `4xl`, `5xl`, `6xl` and `7xl` . It makes line height and spacing smaller as text size grows and also ensures text size is even for better layout.

<Canvas
  of={TextStories.FontSize}
  source={{
    format: true,
    code: `
      <div>
        <Text size="xs">Yorike</Text>
        <Text size="sm">Yorike</Text>
        <Text size="md">Yorike</Text>
        <Text size="lg">Yorike</Text>
        <Text size="xl">Yorike</Text>
        <Text size="2xl">Yorike</Text>
        <Text size="3xl">Yorike</Text>
        <Text size="4xl">Yorike</Text>
        <Text size="5xl">Yorike</Text>
        <Text size="6xl">Yorike</Text>
        <Text size="7xl">Yorike</Text>
      </div>
    `,
  }}
/>

### [Font Weight](#text-font-weight)

To adjust the weight of text, use the `fontWeight` prop.
