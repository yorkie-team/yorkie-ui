## Import

```jsx
import { Grid, GridItem } from 'yorkie-ui';
```

- **_Grid_**: The main wrapper with display: grid.
- **_GridItem_**: Used as a child of Grid to control the span, and start positions within the grid.

## Overview

`Grid` is a type of layout that is used to manage and organize grid structures.

### [Grid Responsive](#grid-responsive)

You can customize the number of columns with breakpoints: `base`, `sm`, `md`, `lg`, and `xl`. Other props also can use breakpoints.

```jsx
  base: '0rem', // 0px
  sm: '40rem', // ~640px - base font size 16px
  md: '48rem', // ~768px
  lg: '64rem', // ~1024px
  xl: '80rem', // ~1280px
```

### [Spanning Columns](#spansning-columns)

To span specific columns or rows in a grid layout, use the `colSpan` and `rowSpan` props in the `GridItem` component. Also, specify the `templateColumns` and `templateRows`.

<Canvas
  of={GridStories.Spanning}
  source={{
    format: true,
    code: `
      <Grid h="200px" gridTemplateRows={'repeat(2, 1fr)'} gridTemplateColumns="repeat(5, 1fr)" gap={4}>
        <GridItem gridRow={2} colSpan={1} bg="accent.default" />
        <GridItem colSpan={2} bg="papayawhip" />
        <GridItem colSpan={2} bg="papayawhip" />
        <GridItem colSpan={4} bg="accent.default" />
      </Grid>
    `,
  }}
/>

### [Starting and ending lines](#spansning-columns)

Pass the `colStart` and `colEnd` props to the GridItem component to define the start and end positions of an element in the grid.

<Canvas
  of={GridStories.StartingEnding}
  source={{
    format: true,
    code: `
      <Grid gridTemplateColumns="repeat(5, 1fr)" gap={4}>
        <GridItem colSpan={2} h="10" bg="tomato" />
        <GridItem colStart={4} colEnd={6} h="10" bg="papayawhip" />
      </Grid>
    `,
  }}
/>

### [Grid Row Start End](#grid-row-start-end)

Pass the `gridRowStart` and `gridRowEnd` props to the `GridItem` component to specify the grid position of an element.

<Canvas
  of={GridStories.StartEndRow}
  source={{
    format: true,
    code: `
      <Grid gap={4} gridRow={3} gridAutoFlow="row">
        <GridItem h="20" bg="tomato" gridRowStart={2} gridRow={2} />
        <GridItem h="20" bg="papayawhip" gridRowEnd={3} gridRow={2} />
        <GridItem h="20" bg="tomato" gridRowStart={1} gridRowEnd={4} />
      </Grid>
    `,
  }}
/>

### [Grid Columns Start End](#grid-columns-start-end)

Use the `gridColumnStart` and `gridColumnEnd` prop of GridItem to position an element in the nth grid position.
