## Overview

An `RadioGroup` allows the user to select one option from a set. The user can only choose one radio selection at a time.

<Canvas
  of={RadioGroupStories.Overview}
  source={{
    format: true,
    code: `

        {options.map((option) => (
          <RadioGroup.Item key={option.id} value={option.id} disabled={option.disabled}>
            <RadioGroup.ItemControl />
            <RadioGroup.ItemText>{option.label}</RadioGroup.ItemText>
          </RadioGroup.Item>
        ))}
      </RadioGroup.Root>
    `,
  }}
/>

## Usage

A `RadioGroup` object consists of some optional properties:
- `size`
- `orientation`
- `disabled` (default `false`)
- `colorPalette`

The following code shows how you can use the `RadioGroup` component:

<Source format={true} code={`
  import * as RadioGroup from '@/components/ui/radioGroup/RadioGroup';

  const options = [
    { id: 'react', label: 'React', disabled: false },
    { id: 'solid', label: 'Solid' },
    { id: 'svelte', label: 'Svelte' },
    { id: 'vue', label: 'Vue' },
  ];

  <RadioGroup.Root size="md" orientation="horizontal" defaultValue="react" colorPalette="red">
    {options.map((option) => (
      <RadioGroup.Item key={option.id} value={option.id} disabled={option.disabled}>
        <RadioGroup.ItemControl />
        <RadioGroup.ItemText>{option.label}</RadioGroup.ItemText>
      </RadioGroup.Item>
    ))}
  </RadioGroup.Root>
`} />

### Orientation

By using the `orientation` prop of `RadioGroup.Root`, you can change the RadioGroup direction. There are two options: `horizonral` and `vertical`.

`vertical` is set by default because it is the easier for users to scan, especially if the labels are more than 3 words or if the listing is dropped.

But you can customize with RadioGroup `horizontal` like the code snippet below:
<Source format={true} code={`
  <RadioGroup.Root orientation="horizontal">
    //
  </RadioGroup.Root>
`} />

### With default option

By default, the RadioGroup doesn't apply a pre-selected option, but you can add `defaultValue` prop to `RadioGroup.Root` like the code snippet below to create pre-selected option:

<Source format={true} code={`
  <RadioGroup.Root defaultValue="react">
    //
  </RadioGroup.Root>
`} />

You can apply RadioGroup with default option only if:

- The default value is a safe and secure.

- You're willing to collect biased data because the user may not care about the default options.

### ColorPalette

You can use the `colorPalette` prop of `RadioGroup.Root` to custom the `Control` color. The color specified in the color system is required.

By default, the RadioGroup uses the `accent` color from the theme.

<Canvas
  of={RadioGroupStories.WithColor}
  source={{
    format: true,
    code: `
      import * as RadioGroup from '@/components/ui/radioGroup/RadioGroup';

      const options = [
        { id: 'react', label: 'React' },
        { id: 'solid', label: 'Solid' },
        { id: 'svelte', label: 'Svelte' },
        { id: 'vue', label: 'Vue' },
      ];

      <RadioGroup.Root colorPalette="red" defaultValue="react">
        {options.map((option) => (
          <RadioGroup.Item key={option.id} value={option.id} disabled={option.disabled}>
            <RadioGroup.ItemControl />
            <RadioGroup.ItemText>{option.label}</RadioGroup.ItemText>
          </RadioGroup.Item>
        ))}
      </RadioGroup.Root>
    `,
  }}
/>

If you want to custom the `Text` color, add `color` prop of `RadioGroup.ItemText` like the code snippet below:
<Source format={true} code={`
  <RadioGroup.ItemText color="red">{option.label}</RadioGroup.ItemText>
`} />
