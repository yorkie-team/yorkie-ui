## Usage

A `RadioGroup` consists of some properties:
- `size`
- `orientation`
- `disabled` (default `false`)
- `colorPalette`

The following code shows how you can use the `RadioGroup` component:

<Source format={true} code={`
  const options = [
    { id: 'react', label: 'React', disabled: false },
    { id: 'solid', label: 'Solid' },
    { id: 'svelte', label: 'Svelte' },
    { id: 'vue', label: 'Vue' },
  ];

  <RadioGroup.Root size="md" orientation="horizontal" defaultValue="react" colorPalette="red">
    {options.map((option) => (
      <RadioGroup.Item key={option.id} value={option.id} disabled={option.disabled}>
        <RadioGroup.ItemControl />
        <RadioGroup.ItemText>{option.label}</RadioGroup.ItemText>
      </RadioGroup.Item>
    ))}
  </RadioGroup.Root>
`} />

### ColorPalette

Using the `colorPalette` prop to `RadioGroup.Root` custom the `Control` color based on theme colors.

<Canvas
  of={RadioGroupStories.WithColor}
  source={{
    format: true,
    code: `
      const options = [
        { id: 'react', label: 'React' },
        { id: 'solid', label: 'Solid' },
        { id: 'svelte', label: 'Svelte' },
        { id: 'vue', label: 'Vue' },
      ];

      <RadioGroup.Root colorPalette="red" defaultValue="react">
        {options.map((option) => (
          <RadioGroup.Item key={option.id} value={option.id} disabled={option.disabled}>
            <RadioGroup.ItemControl />
            <RadioGroup.ItemText>{option.label}</RadioGroup.ItemText>
          </RadioGroup.Item>
        ))}
      </RadioGroup.Root>
    `,
  }}
/>

If you want to custom the `Text` color, add `color` prop to `RadioGroup.ItemText` like this.

<Source format={true} code={`
  <RadioGroup.ItemText color="red">{option.label}</RadioGroup.ItemText>
`} />
