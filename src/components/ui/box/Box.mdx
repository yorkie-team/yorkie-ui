## Import

```jsx
import { Box } from 'yorkie-ui';
```

The Box component is a useful tool as it helps with three common use cases.

- Firstly, it makes it easier to create responsive layouts.
- Secondly, it provides a shorter way to pass styles through props (such as using `bg` instead of `backgroundColor`).
- Finally, it allows for component composition and overrides through the "as" prop.

## Overview

Box is the fundamental building block of all other UI components. By default, it renders a `div` element.
