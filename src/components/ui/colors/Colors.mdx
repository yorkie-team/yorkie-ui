## Overview

The color code used for props with color-changing properties. For example: `bg`, `color`, `borderColor`, etc...

- [Accent Color](#accentcolor), [Function Color](#functioncolor), [Border Color](#bordercolor)

import { Box } from 'yorkie-ui';

export default function Home() {
  return (
    <Box color="gray.3" bg="orange.3">
      Hello Yorkie
    </Box>
  );
}

## Dark Light Theme

Use dark mode theme we need to add attributes `data-theme`

```html
<html data-theme="dark"></html>
```

and just use color with name have support light/dark theme.

```js
<Box color="orange.2" bg="orange.3">
  Hello Yorkie
</Box>
```
