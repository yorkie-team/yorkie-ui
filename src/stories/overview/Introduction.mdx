# Get started

Yorkie UI is a reusable component library that helps Yorkier contributors build UIs faster.
The goal is to make building durable UIs more productive and satisfying.

## Install

UI components are written in React. It requires Storybook version 8.0 and up.

Add Yorkie UI to your project.

```
npm install --save yorkie-ui
```

### **Use**

Import components you want into your UI

```jsx
import { Button, Badge } from 'yorkie-ui';
```

and use them like so

```jsx
const example = () => (
  <div>
    <Button appearance="primary" onClick={() => 0}>
      Do something
    </Button>
    <Badge onClick={this.clear}>+1</Badge>
  </div>
);
```
